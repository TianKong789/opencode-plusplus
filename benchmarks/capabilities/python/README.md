# Python Capability Benchmark

Evaluates model proficiency in Python coding tasks.

## Task Categories

- Code generation
- Bug fixing
- Refactoring
- Testing
- Type hints
- Async/await patterns

## Scoring

- **Correctness**: Does the code run and produce correct output?
- **Idiomatic**: Does it follow Python best practices?
- **Efficient**: Is the solution reasonably performant?
