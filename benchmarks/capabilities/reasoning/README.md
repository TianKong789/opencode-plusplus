# Reasoning Capability Benchmark

Evaluates model proficiency in logical reasoning and problem-solving.

## Task Categories

- Logical deduction
- Causal reasoning
- Analogical reasoning
- Constraint satisfaction
- Multi-step reasoning
- Edge case analysis

## Scoring

- **Correctness**: Is the conclusion correct?
- **Reasoning**: Is the logic sound and well-explained?
- **Completeness**: Are all cases considered?
