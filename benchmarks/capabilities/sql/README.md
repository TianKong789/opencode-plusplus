# SQL Capability Benchmark

Evaluates model proficiency in SQL query writing.

## Task Categories

- Simple SELECT queries
- JOINs and aggregations
- Window functions
- CTEs and subqueries
- Performance optimization
- Schema design

## Scoring

- **Correctness**: Does the query return expected results?
- **Efficient**: Is the query performant?
- **Readable**: Is the SQL well-structured?
