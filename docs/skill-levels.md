# Skill Levels

## Engineering

Purpose:
Implement software.

Examples:

- Python Coding
- Testing
- Refactoring

---

## Architecture

Purpose:
Improve system design.

Examples:

- Milestone Review
- Architecture Review

---

## Governance

Purpose:
Protect long-term system integrity.

Examples:

- Architecture Guardian
- Release Guardian
- Evolution Guardian

Governance skills never automatically modify code.
They review, evaluate, and approve or reject changes.
